# カピバラ帝国記 — 長芋戦役

森で生まれたグラサンカピバラ「カピノブ」が長芋族に敗北し、仲間を集め、帝国を築き、六つの戦域を巡る全45章の3DアクションRPGです。メインストーリーの想定プレイ時間は、探索・育成を含めて約28〜32時間です。

アップロードされた2枚の原画を基に、カピバラと長芋をプロシージャル3Dモデルとして再構成しています。登場キャラクターの外見はこの2種のみで、名前・体格・装飾によって王、仲間、兵士、ラスボスを判別します。

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 操作

- `WASD` / 矢印キー: 移動
- `Shift`: 疾走
- `Space`: 頭突き攻撃
- `Q`: 帝王突進（覇気を50消費）
- `E`: 会話・勧誘・前哨地制圧
- マウスドラッグ: カメラ回転
- マウスホイール: ズーム
- `Esc`: メニュー

スマートフォン・iPadでは大型の仮想スティックと「攻撃・必殺・話す」ボタンを使用できます。スティックを端まで倒すと自動で疾走し、攻撃ボタンは押しっぱなしでも連続攻撃できます。進行状況はブラウザに自動保存されます。

確定敗北イベント後は、地域の順番に関係なくフィールド上の通常長芋すべてと戦えます。先の地域を先に攻略してもクエスト進行へ反映され、最初に敗北したヤマジとも後から再戦できます。

## ゲーム進行

1. はじまりの森で誕生し、木漏れ日の道を進む
2. 長芋武者ヤマジに襲われ、避けられない敗北を経験
3. 打倒・長芋族を誓い、体格の異なる3名のカピバラを勧誘
4. 木漏れ日の谷、黄金湿原、黒土前線を巡って成長
5. 長芋族の大軍を突破し、黒土砦で大長老ナガトロと決戦（第一部完）
6. 建国を狙う残党軍を倒し、できたてのカピバラ帝国を守る
7. 白根海道を解放し、王都を揺らす「ねばり月の内乱」を止める
8. 地底根城と空中棚田へ遠征し、六つの戦域を帝国へつなぐ
9. 長芋族の始祖が起こす大ねばり戦争を越え、45章目の最終決戦へ

長編部分は6部・36任務で構成されます。通常戦、精鋭戦、章ボス戦が進行に合わせて生成され、名前と体格の異なる長芋が継続的に出現します。既存セーブが大長老撃破後の場合は、そのまま第二部から再開できます。

## Renderへ公開

このリポジトリはRender Static Site用の `render.yaml` を含みます。データベースや環境変数は不要です。

1. GitHub、GitLab、またはBitbucketのリポジトリへpush
2. Render Dashboardで「New Blueprint」を選択
3. このリポジトリを接続して「Apply」

Renderは `npm ci && npm run build` を実行し、`dist` をCDNから配信します。push後の変更は自動デプロイされます。
